# 🔗 Linktree Pessoal

Este é um projeto de página estilo Linktree desenvolvido para centralizar meus principais links profissionais e redes sociais em um único lugar.

## 🚀 Sobre o projeto

A proposta deste projeto é oferecer uma interface simples, moderna e responsiva para facilitar o acesso ao meu portfólio, GitHub, LinkedIn e outros canais.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## 🎨 Layout

O design segue uma abordagem minimalista com tema escuro, buscando uma aparência moderna e agradável, alinhada ao perfil de desenvolvedor.

## 📂 Estrutura do projeto

```
📁 projeto
 ├── 📁 assets
 ├── 📁 arquives
 ├── index.html
 └── style.css
```

## 📄 Funcionalidades

* Exibição de links personalizados
* Redirecionamento para redes sociais
* Download de currículo
* Layout responsivo
